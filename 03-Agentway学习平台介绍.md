### 为什么需要 Agentway？

现在网上关于 AI Agent 的免费资源非常多：各种教程、开源仓库、YouTube 视频、Prompt 技巧…… &#x20;

但大多数人学完之后依然卡在「会用别人做的 Agent」，却很难自己搭出一个**能在工作中真正交付、稳定运行的 Agent 系统**。

Agentway 解决的正是这个痛点。它不是一个散乱的教程合集，而是一条**清晰、连贯、可落地**的核心学习路径（Core Path），让你系统性地从 0 到 1 构建出靠谱的 Agent PoC。



### Agentway 的学习路径设计



平台把学习分为 **4 个主轨道（Track）**，一步步带你升级：

* **Track 0：Preparation**

补齐 TypeScript 和工程基础，让后续开发更顺畅（可选）。

* **Track 1：Awakening（唤醒）**

掌握 Agent 循环、工具调用、Prompt 模式，跑通你的第一个 Agent。

* **Track 2：Forging（锻造）**

加入记忆机制、评估系统、系统设计模式，让 Demo 变成真正能用的原型。

* **Track 3：Expedition（远征）**

深入生产级话题：多 Agent 协作（MCP）、Agentic RAG、人机协同（HITL）、生产运维等。

学完核心路径，你会完成 **3 个 Capstone 大项目**：

1. 构建一个可复用的 Agent 框架；

2. 开发一个服务真实团队工作流的 Agent 系统；

3) 打造一个更复杂的领域专用 Agent 系统。

除此之外，还有 **6 个 Specials 深度专题**（工业级 RAG、Agent 专用 SFT、LLM Serving、安全、GraphRAG、多模态 & GUI Agent 等），供你在遇到生产难题时进阶使用。



### 学习体验亮点

* 内置 **Chat** 助手，边学边问，快速解卡；

* **138 张智能闪卡**（SM-2 间隔重复算法），帮你牢牢记住核心模式；

* 可视化学习目录 + 进度追踪，清晰看到自己的成长路径；

* 大量动手练习 + 真实案例拆解（包括 Claude Code、OpenClaw 等）。



### 给不同角色同学的收获

* **如果你是开发小白或想系统入门 Agent**：跟着 Core Path 走，你会从「只会调用现成工具」逐步进化成能独立设计、实现和调试 Agent 的**开发者专家**。

* **如果你想成为产品专家**：通过 Capstone 项目，你会学会如何把 Agent 真正嵌入团队工作流，理解人机协同、评估机制、生产落地等产品级思考方式。

* **无论你是开发者还是产品/技术负责人**：学完后，你不再是单纯的「Agent 使用者」，而是能**自主构建、生产级交付 Agent 系统**的人，这在当下 AI 工程化浪潮中，是非常稀缺且高价值的能力。

它特别适合已经有一定编程基础、希望在 AI Agent 方向实现**从使用者到构建者、从实验到交付**进化的人。
