## &#x20;常见问题排查与解答

1. 购买的话：访问 `https://fk.codesome.cn/` 购买；V3 访问 `https://v3.codesome.cn` 使用，二合一访问 `https://v5.codesome.cn` 使用；（V3 兑换码在兑换区兑换）

2. **兑换码和 API key 区分：** Mens 发给大家的一长串的是兑换码，伙计们。不是 api key，api key 自己创建

3. **购买链接：** 购买入口统一使用 `https://fk.codesome.cn/`。

4. **模型支持说明：** Claude 模型和具体版本取决于产品线、分组及当天上游可用性，以后台模型列表和调用结果为准，不要只按客户端下拉列表判断。

5. **GPT-5.6 模型 ID：** 日常任务优先使用 `gpt-5.6-terra`；高难度任务使用 `gpt-5.6-sol`，简单任务使用 `gpt-5.6-luna`。不要填写裸 `gpt-5.6`，它会默认指向更贵的 Sol。

6. **Fable5 当前边界：** 客服于 2026 年 7 月 15 日明确，2026 年 7 月 19 日之后无法使用 Fable，不影响其他模型。不要继续按旧教程创建 Fable 配置或尝试调用；如果未来后台重新显示该模型，再以当天后台模型列表、分组状态和平台公告为准。

1) **使用中的报错排查：** 网络报错、502 等运行中问题统一整理到了[ codesome｜使用问题速查](https://oxv18tgb72z.feishu.cn/docx/MYFSdju3yoDpxAxmzepcT2SgnPf)；上下文自动压缩配置问题请参考[ Claude Code 上下文自动压缩配置](https://oxv18tgb72z.feishu.cn/docx/Y5pPdu1tzowi9bx8bMHcCLr9ny4)。这里保留说明类常见问题，不重复展开。
